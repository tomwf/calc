## How To Use

```bash
https://github.com/tomwf/calc.git
```
Navigate into the directory:
```bash
cd calc
```
Install the dependencies:
```bash
npm i
```
Run the local dev environment:
```bash
npm start
```
Open your browser and visit:
```
http://localhost:3000
```
